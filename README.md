# 技能五子棋学校

一款以 15×15 五子棋为底层规则、以荒诞技能和舞台事故为表层演出的纯前端喜剧策略游戏。玩家可以完成四章剧情、与 AI 自由对战、本地双人，或进入“据说有人只是想正常下棋”的纯净模式。

## 玩法

- 黑棋先手，横、竖或斜线连续五颗同色棋子获胜。
- 自由对战可以分别选择玩家黑方和 AI 白方；本地双人可以分别选择玩家一和玩家二。
- 普通落子获得 1 点技能值（上限 6）；技能按配置消耗技能值且每局通常限用一次。
- 技能围绕“物理招式 2.0”重构：扔棋子、冻时间、摔棋盘、调走对手和展示黑历史都拥有明确结算规则。
- 张呈使用“擒拿”反制飞沙走石，使用“东山再起”捡回被摔坏的棋盘，并能用“时光倒流”撤销上一行动或阻止终结技。
- “力拔山兮”“调呈离山”“两极反转”属于终结技，第8回合后才可发动，并优先进入反制窗口。
- “静如止水”冻结棋盘操作5秒，倒计时结束后由原本应行动的玩家继续，不会吞掉回合。
- “保洁上门”使用固定随机种子清除约一半未保护棋子，相同种子产生相同结果。
- 快乐值满后进入三回合“快乐时刻”，技能消耗降低，观众开始挥手。
- 环境事件使用固定种子伪随机数，确保同一种子下结果可复现。

## 本地运行

需要 Node.js 22.13 或更高版本。

```bash
npm install
npm run dev
```

测试与生产构建：

```bash
npm test
npm run build
npm run preview
```

## 技术结构

- `src/game/types.ts`：完整游戏数据类型与状态机阶段。
- `src/game/config.ts`：四名角色、十三项技能、分类、终结条件与剧情章节。
- `src/game/rules.ts`：棋盘、胜负、技能目标与确定性规则纯函数。
- `src/game/state.ts`：`useReducer` 状态机、技能结算、环境事件与资源流。
- `src/game/ai.ts`：直接胜负、威胁阻挡与启发式候选评估。
- `src/components/BoardCanvas.tsx`：Canvas 棋盘与状态/特效绘制。
- `src/audio.ts`：Web Audio API 原创合成音效。
- `tests/rules.test.ts`：规则、技能、反制、AI 与重开状态测试。

设置、剧情进度和最近战绩保存在 `localStorage`。所有人物、棋盘、特效和音效都由 CSS、Canvas 与 Web Audio 生成，不加载远程素材。

## 运行范围

当前版本仅用于本地运行，不包含 GPT Sites、GitHub Pages 或其他在线部署配置。仓库只保存源码、测试和本地构建设置。

“力拔山兮”现在进入棋盘损坏终结状态；若张呈及时发动“东山再起”，则从摔棋盘前的快照继续。后续可增加完整棋谱回放、更深层 AI 搜索和更多终结技专属分镜。
